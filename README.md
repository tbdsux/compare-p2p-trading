# compare-p2p-trading

Compare P2P prices from different exchages in real-time.

## Current Exchanges Supported

- Binance
- OKX
- MEXC

## Development

```sh
# Clone the project
git clone https://github.com/tbdsux/compare-p2p-trading.git

# Install dependencies
pnpm install

# Run dev
pnpm run dev
```
